# ng-uml-class

Application for UML class diagram **drawing**

## Installing dependencies
After cloning the repo, cd in the directory
do:
```npm install```
then,
```cd client```
then
`npm install`

## launch client and server
* In one terminal
`npm start`
* In another terminal
`cd client`
then,
`ng serve --open`

## Open app
To test just click [localhost:4200](http://localhost:4200)
